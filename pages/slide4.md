## Penggunaan Record
Perlu diperhatikan bahwa record merupakan tipe data, sama seperti tipe data integer atau string. Oleh karena itu, kita dapat menggunakannya dengan mendeklarasikan variabel bertipe data record tersebut.
```cpp twoslash
int main() {
    NilaiMahasiswa mahasiswa1;
}
```
karena variabel mahasiswa1 bertipe data NilaiMahasiswa, variabel tersebut memiliki member-member namaMahasiswa, nilaiKuis, nilaiUTS, dan nilaiUAS. Untuk mengaksesnya, digunakan notasi dot (.) seperti berikut.
